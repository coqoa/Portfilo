안녕하세요. 저는 최병민입니다.

제가 가장 중요하게 생각하는 세가지는 시간 관리 문제 해결 능력 가치 입니다.

개발 공부를 하며 시간 관리를 위한 서비스가 필요했습니다.
원하는 기능을 모두 지원하는 서비스가 없어서 스케줄 관리 웹 앱 'POMOTODO'를 만들었습니다.
공부할 때 많은 도움을 받았고 배포한 지 4개월이 지난현재 약 40명의 이용자에게 서비스를 제공하고 있습니다.

문제 해결 능력을 키우기 위해 항상 '문제의 본질 파악 - 가설 수립 - 검증'하는 과정을 거쳤습니다.
위 과정을 반복하며 문제에 접근하는 방법, 한번 더 생각해보는 습관, 해결책을 찾는 방법들을 익힐 수 있었고 프로젝트를 하는데 많은 도움이 되었습니다.

나를 가장 객관적으로 볼 수 있으며 발전을 위해서 가치를 매우 중요하게 여깁니다.
제가 지닌 '가치'가 어느정도인지그리고어떤 '가치'를 제공할 수 있을지 항상 고민합니다.

효율 중심의 워크플로우 를 지향하며 의견을 제시할 수 있는 사람이 되고자 합니다.

POMOTODO


로컬에서는 정상작동하는데 도메인에서 문제가 생기는 날짜 변경


밤 12시가 되면 날짜가 바뀌면서 DB에 새로운 필드를 생성하도록 해놨는데 도메인에서 정상작동하지 않는 문제가 발생했습니다.
로컬 컴퓨터와 서버 컴퓨터의 Timezone이 달라서 발생한 것이었고, 홈 접속 시 DB에 해당 유저의 당일 데이터가 없다면 생성해 주고 있다면 불러오도록 해서 문제를 해결했습니다.

Expo


여러 번 출력 시 오디오가 뻗어버리는 이슈


여러 번 오디오를 재생하다 보면 모든 오디오가 안 나오는 상황이 발생했습니다.
'unloadAsync해주는 과정'이 없어서 발생한 문제였습니다.
오디오 재생 함수 호출할 때마다 생성, 실행 한 뒤에 setTimeout을 이용해 일정 시간 뒤 unloadAsync(삭제) 해주도록 구현해서 문제를 해결했습니다.
